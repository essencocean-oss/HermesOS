# HermesOS — Agent OS Marketplace

Alpha build for the agent OS layer on top of Hermes Agent.

## What this is
- Multi-tenant Hermes runtime (one container per user)
- Skill registry and marketplace
- Telegram onboarding
- Fiat + crypto billing ready

## Alpha status
- Spawn skill scaffold: install / run / stop / status / kill
- Launch skills: onboarding, telegram-poster, solana-checkout, daily-brief
- Docker-per-user template in progress
