# OceanOS

**OceanOS is the desktop runtime for Hermes Agent.**
It’s one app for install, configuration, chat, automation, agent management, and the public registry/marketplace layer — so users can install skills, agents can run apps, and the ecosystem becomes usable for everyone.

Instead of managing the CLI by hand, OceanOS gives you a guided first-run install, a polished workspace, and a Mission Control surface for skills, agents, schedules, memory, tools, models, profiles, and more.

> **Long-term goal:** build an open marketplace and runtime where anyone installs Hermes skills, publishes agents, and runs AI apps from a single desktop/registry layer.
